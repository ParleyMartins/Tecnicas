Tecnicas
========

So the program can be corrected compiled, the following steps must be taken (only before the first compilation):
	1. Add the libs: jcalendar-1.4.jar and mysql-connector-java-5.1.22-bin.jar to the Project Libraries (click with the right button on JRE System Library > Build Path > Configure Build Path > Add External Jars > (folder libs from the repo)
	2. Install mysql
	3. Type mysql -u 'testuser' -p
	4. Type the password: password
	6. Execute the file BD_CreateCode.sql (in documents folder) on terminal (you can copy and paste it on terminal)
	7. Compile the program and enjoy it.
