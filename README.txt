Tecnicas
========

Repositorio do Trabalho de Tecnicas de Programacao 2014.1

Para que o programa possa ser compilado os seguintes passos devem ser realizados antes da primeira compilação:
	1. Adicionar as libs: jcalendar-1.4.jar e mysql-connector-java-5.1.22-bin.jar às bibliotecas do projeto (clicar com o botão direito em JRE System Library > Build Path > Configure Build Path > Add External Jars > (na pasta libs do repo)
	2. Instalar o mysql
	3. Entrar no mysql pelo terminal
	4. digitar mysql -u 'testuser' -p
	5. Entrar com a senha: password
	6. Executar o arquivo BD_CreateCode.sql (na pasta documentos) no terminal (pode ser copiar todo o codigo e colar)
	7. Compile o programa e enjoy it.
